Crea un formulario que permita ingresar pago por hora, horas trabajadas, 
marcar una casilla si trabajo hora extras, en caso de ser asi se debe activar
un cuadro de texto para ingresar la cantidad de horas extras.
El sueldo se calcula multiplicando pago por hora por horas trabajadas.
Las horas extras se calculan multiplicando la hora normal por las 1.35.
El total sueldo mas horas extras se calcula sumando el sueldo mas horas extras.
Mostrar como salida el sueldo, total horas extras y el total general, todo esto dentro de un groupbox.
El formulario debe tener un boton para calcular, uno para limpiar y uno para cerrar el formulario.
