# Trabajos-5to-d2
Descripcion:

La actividad trata del desarrollo de un sistema que permita registrar movimientos ya sean de salida o entrada y que permita generar reportes de estos mismos, este sistema tambien cuenta con validacion de roles con un inicio de sesion, siendo que en base a tu rol te restringue ciertos modulos de este.

Herramientas utilizadas:

C#

.NET Framework

Visual Studio Community 2022
