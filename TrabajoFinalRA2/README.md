Descripcion:

La actividad trata sobre el desarrollo de un sistema de ventas con generación de reportes, en el cual se registran productos, categorías y ventas en una base de datos, y luego se muestra la información mediante formularios y reportes.

El objetivo es administrar los datos (crear, listar, filtrar y actualizar información) y posteriormente generar una factura o reporte de la venta seleccionada, donde se visualicen los detalles de los productos, precios, cantidades y totales calculados automáticamente.

En esencia, la actividad busca aplicar el manejo de bases de datos, programación en C# y uso de reportes para simular el funcionamiento básico de un sistema real de facturación

Herramientas utilizadas:

C#

.NET Framework

Visual Studio Community 2022
